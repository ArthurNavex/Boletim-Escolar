# Boletim-Escolar
Sistema de Boletim Escolar

Projeto criado para treinar lógica de programação, POO e aprender a linguagem C#

Funcionalidades: 
  *Cadastrar Alunos
  *Ver alunos cadastrados 
  *Adicionar notas de cada trimestre 
  *Ver nota final do Aluno

Como executar:
  *É necessário ter o SDK do .Net no computador
  1 - no terminal va até a pasta do projeto
  2 - compile e execute
